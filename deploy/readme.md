# Tensorflow Serving快速部署SavedModel模型
tensorflow serving只支持SavedModel格式的模型文件，而一般训练的时候ckpt的模型文件格式不支持，需要通过转换得到。
## 1.通过docker pull拉取tensorflow serving镜像
```
docker pull tensorflow/serving
```

## 2.启动服务: docker run
+ -t 为容器分配一个伪输入终端， 通常与-i一起使用
+ --rm 容器退出时自动清理容器内部的文件系统
+ -p 端口映射，宿主机port:容器port
+ -v 绑定挂载目录，必须使用绝对路径，宿主机路径:容器路径
+ -e 传递环境变量，设置模型名字MODEL_NAME
```
docker run -t --rm -p 8501:8501 \
  -v ~/python_work/bert-application/fine_tuned/garbledSents-cased/saved_model:/models/garbledSents \
  -e MODEL_NAME=garbledSents \
  tensorflow/serving &
```
